# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |


## Reporting a Vulnerability

Please post vulnerabilities in the "Issues" section. We will check if it is a actual vulnerability and then fix it if it is.
